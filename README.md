# xiaofei-Home-page

一个使用html,过程使用ai开发,后经过我手动修改后的一个主页(引导页或首页)代码,欢迎使用(代码初期有一个叫江江的朋友给我写了开头,后面不太想麻烦他就自己琢磨来写)

# 初衷

说实话我一个英文都看不懂的人,这个代码前前后后我修修改改的用了几个月了,每次都是让群里的小伙伴使用后提出意见我再去改,说实话又艰难又充满了挑战,来开源它的时候我也是一步一步翻译GitHub才搞定,希望大佬们嘴下留情别喷我这没学过英文的人,我的初衷只是为了希望能为开源之路做出一些贡献

## 自适应

   代码做了自适应处理,能完美兼容各种设备的运行
   
## QQ和微信遮罩

   给qq和微信增加了遮罩功能,避免被QQ微信给拦截
   
## 夜间和白天主题

  主题采用了根据ip日升月落来判定,自动启动白天或者夜间模式,也可以在右上角进行手动切换
